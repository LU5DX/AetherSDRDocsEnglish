AetherSDR

The user manual

AetherSDR brings FlexRadio operation to Linux, macOS, and Windows — without Wine or virtual machines. Built with Qt6 and C++20, it speaks the SmartSDR protocol natively. Works with any FlexRadio transceiver running SmartSDR firmware v3.x or v4.x.
