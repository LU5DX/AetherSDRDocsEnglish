# Open the log folder to grab multiple files

Use this page to open the log directory in your OS file browser so you can copy, attach, or archive multiple log files at once.

## Before you start

- Open AetherSDR.
- No radio connection is required.

## Steps

1. Click `Help > Support...` to open the Support & Diagnostics dialog.
2. Click `Open Log Folder`.

The log directory opens in your OS file browser. From there you can select, copy, or attach files as needed.

## What each control does

| Control | Behavior |
|---|---|
| **Open Log Folder** | Opens the log directory in the OS file browser. |
| **Log path label** | Shows the full path to the current log file, so you know which directory will open. |
| **Log file size** | Shows the current size of the active log file. |

## Tips

- The log path label in the dialog shows the exact directory that `Open Log Folder` will open. Check it first if you are unsure where logs are stored.
- If you want to capture a clean reproduction of a bug, clear the log before reproducing the issue, then open the folder to retrieve the resulting file.
