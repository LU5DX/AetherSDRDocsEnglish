# Reset every shortcut back to defaults

Use this procedure to restore all keyboard shortcut bindings to their factory defaults in a single operation. This is useful when you have made many custom bindings and want to start over.

## Before you start

- No radio connection is required.
- Any custom bindings you have made will be permanently replaced by the defaults. There is no undo.

## Steps

1. Open `View > Configure Shortcuts...`.
2. Click `Reset All to Defaults` in the bottom-left corner of the dialog.
3. When the confirmation prompt appears, click `Yes`.
4. Click `Close`.

## What each control does

| Control | Behavior |
|---|---|
| `Reset All to Defaults` | Resets every binding to its default key. A confirmation prompt appears before any changes are applied. |
| `Close` | Closes the dialog. Changes take effect immediately; there is no separate Save step. |

## Tips

- To check what the default key for a specific action is before resetting, consult the **Default Key** column in the action table.
- If you only want to reset a single shortcut rather than all of them, use `Reset to Default` after selecting the relevant key on the keyboard map.
