# Clear every spot from the panadapter

Use this page to instantly remove all DX spots currently displayed on the panadapter. This is useful when the spot list has grown cluttered or stale and you want a clean view without waiting for spots to expire.

## Before you start

- Open the Spot Settings dialog by right-clicking the spots overlay on the panadapter and selecting the spot settings option from the context menu.

## Steps

1. Open the Spot Settings dialog from the panadapter context menu.
2. Click **Clear All Spots**.

All spots are removed from the panadapter immediately. New spots arriving from your configured sources will appear normally after the clear.

## Tips

- Clearing spots does not change the `SpotsLifetime` setting. Incoming spots will continue to appear and expire according to the current lifetime value.
- The **Total Spots:** indicator in the dialog shows the live count of tracked spots. After clicking **Clear All Spots**, this count will drop to zero and then climb again as new spots arrive.
- If you want to stop spots from reappearing at all, use the **Spots:** toggle to set it to Disabled rather than clearing. See [Turn spots on or off](turn-spots-on-or-off.md).
