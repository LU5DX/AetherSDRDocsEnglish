# Configure UI Scale

`View > UI Scale` lets you increase or decrease the size of the AetherSDR interface. Use this to match your display resolution or personal preference.

## Before you start

- AetherSDR must be running.

## Steps

1. Click `View` in the menu bar.
2. Hover over `UI Scale` to open the submenu.
3. Select one of the fixed scale radio buttons: `75%`, `85%`, `100%`, `110%`, `125%`, `150%`, `175%`, or `200%`. The interface rescales immediately.

Alternatively, use the keyboard shortcuts without opening the menu:

- **Zoom In:** `Ctrl+=`
- **Zoom Out:** `Ctrl+-`
- **Reset:** `Ctrl+0`

## What each control does

| Control | Description | Default |
|---|---|---|
| `75%` – `200%` | Fixed scale radio buttons. Selecting one sets the UI to that percentage of its base size. | `100%` |
| Zoom In (`Ctrl+=`) | Increases scale one step at a time through the available percentages. | — |
| Zoom Out (`Ctrl+-`) | Decreases scale one step at a time through the available percentages. | — |
| Reset (`Ctrl+0`) | Returns the UI scale to `100%`. | — |

## Tips

- `Ctrl+0` is the fastest way to return to the default scale if the interface becomes too large or too small to navigate comfortably.
- Zoom In and Zoom Out step through the same fixed values as the radio buttons, so the scale always lands on a defined percentage.
