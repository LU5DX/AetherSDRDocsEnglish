# Enable Blink Status Indicator

`View > Blink Status Indicator` controls whether the title bar status indicator blinks to signal the radio heartbeat. Disabling it stops the blinking while keeping the indicator visible.

## Before you start

- AetherSDR must be running and connected to a Flex radio to observe the heartbeat blink.

## Steps

1. Click `View` in the menu bar.
2. Click `Blink Status Indicator` to toggle it.

A check mark next to the item means blinking is enabled. Selecting it again removes the check mark and stops the blink.

## What each control does

| Control | Default | Description |
|---|---|---|
| `Blink Status Indicator` | On (checked) | When checked, the title bar indicator blinks to signal radio heartbeat. When unchecked, the indicator is static. |

## Tips

- If the title bar activity is distracting during extended operating sessions, uncheck `Blink Status Indicator` to stop the blink without affecting the radio connection.
