# Getting Started

`Help > Getting Started...` opens the built-in beginner guide for connecting to and using AetherSDR with a FLEX-8600 radio.

## Before you start

- AetherSDR is installed and running on your system.

## Steps

1. Click **Help** in the menu bar.
2. Click **Getting Started...**.

The beginner guide opens in a new window.
