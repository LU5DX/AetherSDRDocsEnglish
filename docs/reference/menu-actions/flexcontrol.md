# FlexControl

`Settings > FlexControl...` opens the serial port configuration for the FlexControl hardware VFO knob. Use this page to assign the serial port that AetherSDR uses to communicate with a FlexControl device.

## Before you start

- Connect your FlexControl hardware to the computer before opening this dialog.
- AetherSDR must be built with serial port support (`HAVE_SERIALPORT`). If `FlexControl...` does not appear in the `Settings` menu, your build does not include this feature.

## Steps

1. Click `Settings > FlexControl...`.
2. The Radio Setup dialog opens on the Serial tab.
3. Select the serial port corresponding to your FlexControl device from the available port list.
4. Click `OK` to save the configuration.
