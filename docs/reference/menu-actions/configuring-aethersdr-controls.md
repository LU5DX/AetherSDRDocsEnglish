# Configuring AetherSDR Controls

This page describes what the `Help > Configuring AetherSDR Controls...` menu item does. It opens an in-application guide covering how to customize applets, keyboard shortcuts, and MIDI mappings.

## Before you start

- AetherSDR must be running and connected to a FLEX-8600 radio, or at minimum launched to the main window.

## Steps

1. Click `Help` in the menu bar.
2. Click `Configuring AetherSDR Controls...`.

The guide opens, describing how to customize the applet panel, configure keyboard shortcuts, and set up MIDI controller mappings.
