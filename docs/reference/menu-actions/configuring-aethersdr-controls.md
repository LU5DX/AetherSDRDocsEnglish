# Configuring AetherSDR Controls

This page describes how to open the built-in guide for customizing applets, keyboard shortcuts, and MIDI mappings in AetherSDR.

## Before you start

- AetherSDR must be running and connected to a FLEX-8600 radio.

## Steps

1. Click `Help` in the menu bar.
2. Click `Configuring AetherSDR Controls...`.

The guide opens and covers three areas of customization: applet arrangement, keyboard shortcut bindings, and MIDI controller mappings.
