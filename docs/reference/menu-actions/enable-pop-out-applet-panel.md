# Enable Pop Out Applet Panel

Use `View > Pop Out Applet Panel` to detach the applet panel from the main window into its own floating window, or to dock it back into the main window.

## Before you start

- The applet panel must be visible. If it is hidden, enable it first via `View > Applet Panel`.

## Steps

1. Click `View` in the menu bar.
2. Click `Pop Out Applet Panel`.
   The applet panel detaches into a separate floating window. A checkmark appears next to the menu item when the panel is popped out.
3. To dock the panel back into the main window, click `View > Pop Out Applet Panel` again.

## What each control does

| Control | Description | Shortcut | Default |
|---|---|---|---|
| `Pop Out Applet Panel` | Checkable. Floats the applet panel into a separate window when checked; docks it back into the main window when unchecked. | `Ctrl+Shift+S` | Docked (unchecked) |

## Tips

- You can also toggle the pop-out state with the keyboard shortcut `Ctrl+Shift+S` without opening the menu.
- Moving or resizing the floating window does not affect the main window layout.
