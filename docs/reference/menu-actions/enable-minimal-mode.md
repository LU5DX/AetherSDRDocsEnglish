# Enable Minimal Mode

Minimal Mode hides the title bar and controls to give the panadapter the maximum available screen area. Use it when you want a clean, uncluttered view of the spectrum.

## Before you start

- AetherSDR must be running and connected to a FLEX-8600 radio.

## Steps

1. Click `View` in the menu bar.
2. Click `Minimal Mode`.

The item is checkable. A check mark next to `Minimal Mode` means it is active. Clicking it again removes the check mark and restores the title bar and controls.

You can also toggle Minimal Mode at any time with the keyboard shortcut **Ctrl+M**.

## Tips

- If you also want to remove the window frame entirely, see [Enable Frameless Window](enable-frameless-window.md).
- To recover hidden controls when the menu bar is not visible, press **Ctrl+M** to toggle Minimal Mode off.
