# Enable Minimal Mode

Minimal Mode hides the title bar and controls to maximize the panadapter area. Use it when you want the largest possible view of the spectrum display.

## Before you start

- AetherSDR must be running and connected to a radio.

## Steps

1. Click `View` in the menu bar.
2. Click `Minimal Mode`.

The menu item is checkable. Click it again to restore the title bar and controls.

## Tips

- The keyboard shortcut `Ctrl+M` toggles Minimal Mode without opening the menu.
- To also remove OS window decorations, enable `View > Frameless Window` (`Ctrl+Shift+F`) alongside Minimal Mode.
