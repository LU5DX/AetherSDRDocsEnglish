# Contributing to AetherSDR

This page describes how to open the contributing guide, which covers submitting code, bug reports, and feature requests for AetherSDR.

## Before you start

- AetherSDR must be running.

## Steps

1. Click `Help` in the menu bar.
2. Click `Contributing to AetherSDR...`.

The contributing guide opens. Follow the instructions there to submit code contributions, file bug reports, or propose feature requests.
