# Contributing to AetherSDR

This page describes how to open the contribution guide from within AetherSDR. The guide covers submitting code, reporting bugs, and requesting features.

## Before you start

- AetherSDR must be installed and running.

## Steps

1. Click `Help` in the menu bar.
2. Click `Contributing to AetherSDR...`.

The contribution guide opens, covering how to submit code contributions, file bug reports, and request new features.
