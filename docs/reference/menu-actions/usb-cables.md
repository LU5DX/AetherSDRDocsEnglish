# USB Cables

`Settings > USB Cables...` opens the USB Cables tab of the Radio Setup dialog, where you assign USB cable connections for CAT control and audio routing.

## Before you start

- AetherSDR must be connected to a Flex radio.
- The USB cable must be physically connected between the radio and your computer before assigning it here.

## Steps

1. Click `Settings > USB Cables...`.
2. The Radio Setup dialog opens at the USB Cables tab.
3. Configure CAT and audio assignments for each connected USB cable using the controls in the tab.
4. Close the dialog when finished.
