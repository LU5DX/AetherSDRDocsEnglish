# USB Cables

`Settings > USB Cables...` opens the USB Cables tab of the Radio Setup dialog, where you assign USB cable connections for CAT control and audio routing.

## Before you start

- AetherSDR must be connected to a FLEX-8600 radio.
- Any USB cables you want to assign must already be connected to the radio and recognized by it.

## Steps

1. Click `Settings > USB Cables...`.
2. The Radio Setup dialog opens with the USB Cables tab active.
3. Configure your USB cable assignments for CAT and audio as needed.
4. Close the dialog when finished.
