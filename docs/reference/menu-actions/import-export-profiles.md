# Import/Export Profiles

This page covers the `Profiles > Import/Export Profiles...` menu item, which is intended for backing up and sharing AetherSDR profiles.

## Before you start

- This feature is not yet implemented. The menu item exists but does not open a functional dialog.

## Steps

1. Click `Profiles > Import/Export Profiles...`.
2. No dialog opens. The feature is not yet implemented.
