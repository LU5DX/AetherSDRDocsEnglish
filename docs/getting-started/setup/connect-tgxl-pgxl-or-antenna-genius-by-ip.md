# Connect TGXL, PGXL or Antenna Genius by IP

This page explains how to manually connect a TGXL, PGXL, or Antenna Genius device to AetherSDR using its IP address. Use this when the device is not auto-discovered on your network.

## Before you start

- AetherSDR must already be connected to the radio. The Peripherals tab is only available while a radio connection is active.
- You need the IP address of the TGXL, PGXL, or Antenna Genius device.

## Steps

1. Open `Settings > Radio Setup...`.
2. Click the **Peripherals** tab.
3. Locate the row for the device you want to connect (TGXL, PGXL, or Antenna Genius).
4. Enter the device's IP address in the field for that device.
5. Click **Connect** for that device.

To disconnect a device, click **Disconnect** for the corresponding row.
