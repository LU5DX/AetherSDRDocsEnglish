# Enlarge or shrink the spot font

Use the **Font Size:** slider in Spot Settings to make spot callsigns and labels larger or smaller on the panadapter.

## Before you start

- At least one panadapter must be open and visible.
- Spots must be active; if the spot overlay is not showing, see [Turn spots on or off](turn-spots-on-or-off.md).

## Steps

1. Right-click anywhere on the panadapter's spot overlay to open the context menu, then select **Spot Settings**.
2. Locate the **Font Size:** row.
3. Drag the slider left to decrease the font size or right to increase it. The numeric readout beside the slider updates immediately.
4. Close the dialog. The new size is saved automatically.

## What each control does

| Control | Behavior | Default | Valid range | Setting key |
|---|---|---|---|---|
| **Font Size:** slider | Sets the text size of spot labels drawn on the panadapter. | 16 | 8 – 32 | `SpotsFontSize` |

## Tips

- The size value is in points. A setting of 16 matches a typical medium label; move toward 8 for dense band conditions where many spots overlap, or toward 32 for easy reading at a distance.
- Font size applies to both DX spots and memory overlays when memories are enabled.
