# Learn the differences between NR2, NR4, DFNR and MNR

AetherSDR includes a bundled help topic that explains the noise reduction modes available in the radio — NR2, NR4, DFNR, and MNR — so you can choose the right algorithm for your operating conditions without needing an internet connection.

## Before you start

- No radio connection is required to read this help topic.
- AetherSDR must be installed and running.

## Steps

1. Click `Help` in the menu bar.
2. Click `Understanding Noise Cancellation...`.
3. Read the topic in the Markdown viewer. Scroll as needed.
4. Click `Close` when finished.

## Tips

- The Help menu keeps each guide separate. You can reopen `Help > Understanding Noise Cancellation...` at any time without disturbing your radio session.
- If you want to tune the NR2, NR4, DFNR, or MNR parameters directly, open `Settings > AetherDSP Settings...`.
