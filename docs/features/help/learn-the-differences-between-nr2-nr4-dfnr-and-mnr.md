# Learn the differences between NR2, NR4, DFNR and MNR

This page explains how to open the bundled noise cancellation guide in AetherSDR, which describes the differences between the NR2, NR4, DFNR, and MNR noise reduction modes available on the FLEX-8600.

## Before you start

- No radio connection is required to read this help topic.
- AetherSDR must be installed with its bundled help assets intact.

## Steps

1. Click `Help > Understanding Noise Cancellation...` in the menu bar.
2. The HelpDialog opens with the title set to the noise cancellation topic.
3. Read the content in the Markdown viewer. The topic covers the differences between NR2, NR4, DFNR, and MNR.
4. To search within the topic, type a term in the Find field and click Next or Previous to step through matches. Press Return to find the next match; press Shift+Return to find the previous match.
5. Click Close when finished.

## Tips

- The Find field border turns red when no matches exist for your search term. Clear the field or revise your term and try again.
- If the help file fails to load, AetherSDR displays a message in the viewer. Reinstall the application to restore missing help assets.
- Each help topic in the Help menu is separate. You can reopen just this topic at any time without navigating away from other guides.
