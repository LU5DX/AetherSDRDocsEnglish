# Search Memories by Name

Use the Search field in the Memory Channels dialog to filter the memory table down to entries whose names match what you type. This is useful when you have a large number of stored frequencies and need to find one quickly.

## Before you start

- AetherSDR must be connected to a radio. The Memory Channels dialog requires an active radio connection.
- Open the Memory Channels dialog via `Settings > Memory...`.

## Steps

1. Open `Settings > Memory...`.
2. Click the **Search:** field at the top of the dialog.
3. Type part or all of the memory name you are looking for. The table filters as you type.
4. To clear the search and show all memories, click the clear button inside the **Search:** field, or delete the text manually.

## What each control does

| Control | Behavior | Notes |
|---|---|---|
| **Search:** | Filters the memory table to rows whose name matches the text you type. | Has a clear button. Press Enter to activate the currently highlighted row. |
| Memory table | Displays the filtered results. Columns include Group, Owner, Frequency, Name, Mode, and others. | Click a row to select it. Double-click a row to tune the radio to that memory. |

## Tips

- The table filters on every keystroke — you do not need to press Enter to see results. Press Enter to tune the radio to the currently highlighted row.
- To search and then act on multiple results, use Shift-click to select a range or Ctrl-click (Command-click on macOS) to add individual rows to the selection.
- Combine **Search:** with the **Profile:** filter to narrow results by both name and profile group at the same time.
