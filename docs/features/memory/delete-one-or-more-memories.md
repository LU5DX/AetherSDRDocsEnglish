# Delete one or more memories

Use the Memory Channels dialog to remove one or more stored memory entries from the radio. Deleted memories are permanently removed after confirmation.

## Before you start

- The radio must be connected. The Memory Channels dialog requires an active radio connection.
- Know which memories you want to delete. Use the Search: field or Profile: filter to narrow the list if needed.

## Steps

1. Open `Settings > Memory...`.
2. In the memory table, select the row or rows you want to delete.
   - Click a single row to select it.
   - Shift-click to select a contiguous range.
   - Ctrl-click (or Command-click on macOS) to add or remove individual rows from the selection.
   - To select every row, click `Select All`.
3. Confirm the selection count shown in the indicator at the bottom right of the dialog (for example, `3 selected`).
4. Click `Remove`.
5. When the confirmation dialog appears, confirm the deletion.

The selected memories are removed from the radio.

## Tips

- Use `Search:` to filter the table by name before selecting, so you can find the target memories quickly without scrolling. See [Search memories by name](search-memories-by-name.md).
- Use `Profile:` to filter by group before clicking `Select All` if you want to delete all memories in a particular profile. See [Filter memories by profile](filter-memories-by-profile.md).
- Before deleting a large set of memories, consider exporting them first as a backup. See [Export memories for backup or sharing](export-memories-for-backup-or-sharing.md).
