# Export memories for backup or sharing

Export one or more memory channels to a CSV file so you can back them up or share them with other operators.

## Before you start

- AetherSDR must be connected to the radio. The Memory Channels dialog requires an active radio connection.
- You must have at least one memory stored. If the memory table is empty, there is nothing to export.

## Steps

1. Open `Settings > Memory...` to open the Memory Channels dialog.
2. Select the rows you want to export. Click a row to select it. Shift-click to select a range. Ctrl-click (Command-click on macOS) to add or remove individual rows from the selection.
3. To export every memory, click `Select All` to select every row.
4. Click `Export...`.
5. In the file dialog that opens, confirm or change the destination path and filename, then save the file. The default filename is based on the current date and time and the application version, saved to your `Documents` folder.

## Tips

- Use the `Profile:` combo box to filter the table to a single group before clicking `Select All`. This lets you export only the memories that belong to a particular profile without manually picking rows.
- Use the `Search:` field to filter by name if you want to review what is in the table before selecting.
