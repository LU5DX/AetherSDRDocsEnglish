# Delete a binding

Remove a single MIDI binding from the bindings table, or clear every binding at once. Use this when a control mapping is no longer needed or you want to reassign a physical knob to a different parameter.

## Before you start

- The MIDI Controller Mapping dialog must be open. If it is not, go to `Settings > MIDI Mapping...`.
- You must have at least one existing binding visible in the bindings table.

## Steps

### Delete a single binding

1. Open `Settings > MIDI Mapping...`.
2. In the bindings table, locate the row for the binding you want to remove.
3. Click `×` in the rightmost column of that row.

The row is removed immediately. The change is saved automatically.

### Delete all bindings at once

1. Open `Settings > MIDI Mapping...`.
2. Click `Clear All`.

Every row in the bindings table is removed. The change is saved automatically.

## What each control does

| Control | Description |
|---|---|
| `×` (delete row) | Removes the single binding on that row. |
| `Clear All` | Removes every binding in the table. |

## Tips

- If you delete a binding by mistake, you can restore it by loading a previously saved profile. See [Load a previously saved MIDI profile](load-a-previously-saved-midi-profile.md).
- Before using `Clear All`, consider saving your current bindings as a profile first. See [Save the current mapping as a named profile](save-the-current-mapping-as-a-named-profile.md).
