# Stop a Playback in Progress

Use this page to stop the Digital Voice Keyer while a slot is playing back on-air. Stopping playback drops the transmission immediately and returns the DVK to the Idle state.

## Before you start

- The radio must be connected. The DVK panel requires an active radio connection.
- A playback must already be in progress. The status indicator at the bottom of the DVK panel reads "Status: Playing".

## Steps

1. Locate the DVK panel in the main window.
2. Click **■ STOP**.

Playback stops immediately. The status indicator returns to "Status: Idle" and the slot's progress bar is hidden.

**Alternative — keyboard:** Press Escape. If no inline rename is active, Escape stops the current playback.

**Alternative — F-key button:** Click the slot's active F-key button (for example, **F1**). Clicking an F-key button while that slot is playing acts as a toggle and stops playback.

## What each control does

| Control | Behavior when playback is active |
|---|---|
| **■ STOP** | Stops recording, playback, or preview — whichever is currently active. |
| **F1 … F8 slot buttons** | Clicking the active slot's button while it is playing stops playback. Clicking a different slot's button starts playback of that slot. |
| Slot progress bars | Visible and advancing during playback; hidden when the DVK returns to Idle. |
| Status indicator | Shows "Idle", "Recording", or "Playing". Returns to "Idle" after stop. |

## Tips

- **■ STOP** works for recording and preview as well as playback. One button covers all three active states.
- Pressing Escape only stops the active operation if no slot rename is currently open. If a rename text field is visible, Escape cancels the rename instead.
