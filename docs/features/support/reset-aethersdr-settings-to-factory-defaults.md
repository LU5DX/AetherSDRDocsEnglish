# Reset AetherSDR Settings to Factory Defaults

Use this procedure to wipe AetherSDR's locally stored settings and NR2 wisdom cache back to their defaults. Radio settings stored on the Flex radio itself are not affected.

## Before you start

- Back up any configuration you want to keep. The reset cannot be undone.
- A radio connection is not required.

## Steps

1. Open `Help > Support...`.
2. Click `Reset Settings`.
3. When the confirmation dialog appears, confirm the reset.
4. Restart AetherSDR for all defaults to take effect.

## Tips

- Only AetherSDR's local settings and the NR2 wisdom cache are deleted. All settings stored on the radio — profiles, antenna ports, band data, and so on — remain intact.
