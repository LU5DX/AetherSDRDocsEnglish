# Clear all spots from the panadapter

Removing all spots at once clears the panadapter overlay and the spot list table without disconnecting any spot sources.

## Before you start

- At least one spot source must have delivered spots, otherwise there is nothing to clear.
- SpotHub must be accessible via `Settings > SpotHub...`.

## Steps

1. Open `Settings > SpotHub...`.
2. Click the **Spot List** tab.
3. Click **Clear All Spots**.

All spots are immediately removed from the panadapter overlay and from the spot list table. Connected sources continue running and will populate new spots as they arrive.

## Tips

- **Clear** on the **Spot List** tab empties only the visible table for the current band filter selection. **Clear All Spots** removes every tracked spot regardless of band filter state.
- Spots return automatically once active sources send new data. To stop new spots arriving, stop or disconnect each source individually before clearing.
