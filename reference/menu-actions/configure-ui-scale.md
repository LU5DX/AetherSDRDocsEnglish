# Configure UI Scale

`View > UI Scale` lets you increase or decrease the size of the AetherSDR interface to suit your display and viewing distance.

## Before you start

- AetherSDR must be running.

## Steps

1. Click `View` in the menu bar.
2. Hover over **UI Scale** to open the submenu.
3. Click one of the fixed scale radio buttons: **75%**, **85%**, **100%**, **110%**, **125%**, **150%**, **175%**, or **200%**. The interface rescales immediately.

Alternatively, use the keyboard shortcuts to adjust scale incrementally:

- **Zoom In** — `Ctrl+=`
- **Zoom Out** — `Ctrl+-`
- **Reset** — `Ctrl+0` (returns to 100%)

## What each control does

| Control | Action | Shortcut |
|---|---|---|
| 75% | Sets UI scale to 75% of default | — |
| 85% | Sets UI scale to 85% of default | — |
| 100% | Sets UI scale to default size | — |
| 110% | Sets UI scale to 110% of default | — |
| 125% | Sets UI scale to 125% of default | — |
| 150% | Sets UI scale to 150% of default | — |
| 175% | Sets UI scale to 175% of default | — |
| 200% | Sets UI scale to 200% of default | — |
| Zoom In | Increases scale one step | `Ctrl+=` |
| Zoom Out | Decreases scale one step | `Ctrl+-` |
| Reset | Returns scale to 100% | `Ctrl+0` |

## Tips

- Use `Ctrl+=` and `Ctrl+-` for quick adjustments without opening the menu.
- **Reset** (`Ctrl+0`) returns the interface to 100% regardless of the current scale.
